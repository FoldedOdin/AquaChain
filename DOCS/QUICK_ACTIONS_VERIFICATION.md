### Settings Flow:
```
User clicks Settings
    ↓
Frontend calls GET /api/admin/settings
    ↓

    ↓
Modal opens with loaded settings
    ↓
User modifies settings
    ↓
User clicks Save
    ↓
Frontend calls PUT /api/admin/settings
    ↓
Backend validates and saves to .dev-data.json
    ↓
Backend returns success
    ↓
Frontend shows success message
    ↓
Modal closes
```

    ↓
